---맨처음 할때--------------------------------------

--로컬에서의 작업---
1. 깃초기화
	git init
2. stage-area
	git add .
3. 버전생성
	git commit -m "first commit"

---원격 저장소 관리----
1. 깃허브 로그인 후 New repository 생성
2. 브랜치명 변경
	git branch -M main
3. 깃 원격 주소 등록
	git remote add origin https://github.com/qwerewqwerew/super-duper-waddle.git
4. 깃 원격저장소로 코드 올리기
	git push -u origin main
